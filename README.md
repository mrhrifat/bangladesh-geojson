## Bangladesh GEO JSON

GeoJSON is a format for encoding a variety of geographic data structures. In this repository I share the GEO JSON of a country named Bangladesh.

## Features/Infomation
- Division && District
  - ID
  - Division ID (Only District)
  - Name
  - Bengali Name
  - Latitude
  - Longitude
